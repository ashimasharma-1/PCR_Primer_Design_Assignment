PCR primer design and specificity validation using NCBI Primer-BLAST.
PCR Primer Design Assignment
----------------------------

Gene Selected:  ACTB 

Objective:
To design a high-quality PCR primer set for a human gene using NCBI Primer-BLAST by following standard primer design parameters and ensuring high specificity against the human genome.

Design Parameters:
- Primer melting temperature (Tm): ~60 °C  
- GC content: 40–60 %  
- Product size: 80–200 bp  
- Specificity check against: Homo sapiens (taxid: 9606)

Selected Primer Pair:
Primer Pair 3 was selected because it satisfied all the primer design criteria and showed high specificity with no significant off-target binding.

Expected PCR Product Size:
106 bp

Tools Used:
